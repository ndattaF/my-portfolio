Website Functionalities

Landing page
When a user accesses the website using a URL, they will see the landing or home page of the restaurant. it provides users with a welcome message from the restaurant. Options to look at the menu and reserve a table are available at multiple places to cater to different user behaviors.
Navigation Section
The navigation bar provides links to the website sections. Hovering over the options on the Navigation Bar or any buttons on the landing page changes the color of the option. This makes it easy for a user to understand where the mouse pointer is or which item has been selected.
About section
This section provides a description, vision, and a brief history of the restaurant. A restaurant's menu is one of the key factors that drive users to the restaurant
menu section
the, a user has the option to look at the complete menu by default. Each menu item has a title, pricing, and description. A user can also look at a specific category, such as beverages, lunch, or dinner. For example, when a user navigates to the beverages section, they will see a list of the available beverages such as juices, smoothies, and coffee. Photos of a restaurant can also influence a user's decision. With that in mind, the Gallery section provides several images of the food and the restaurant's ambiance.
Contact Us section
The Contact Us section provides details such as location map address, open hours, and phone number. Users can also use the form to clarify any questions or to provide feedback.
book a table feature
the book a table feature enables a user to book on a specified date and time for a specified number of guests. The user just needs to complete the form details and click the "Book a Table" button.
footer
The website also has a footer section which provides social media links for the restaurant and links to the Home, About Us, and Contact Us pages
responsiveness
the website is also responsive. For example, notice how the page layouts adjust on a mobile device.
